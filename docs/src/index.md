# ARP Engineering Guide

This book explains how the ARP application works from source code, with short examples taken from the repository.

Scope:

- Desktop app startup and UI frame flow.
- Multi-agent orchestration and conversation execution.
- Run artifacts, event ledger, and chat persistence.
- Python runtime lifecycle and task tracing.
- Security controls and HTTP policy.
- Environment configuration.

Read this guide in order if you are onboarding. Use the Configuration chapter as quick reference during operations.
